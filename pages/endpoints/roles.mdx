# Roles Collection

The **Roles Collection** manages user roles and permissions within the system, allowing administrators to create, update, delete roles, and assign or remove roles and permissions for users. Below are the detailed descriptions for each endpoint in this collection.

---

## **Create Role**

**Method**: `POST`  
**URL**: `/roles/create-role`

Creates a new role in the system.

### Request

```json
{
  "name": "basic",
  "description": "The most basic role",
  "permissions": ["auth.accounts.list"]
}
```

### Response

```json
{
  "message": "Role created successfully"
}
```

## Update Role

**Method**: `POST`  
**URL**: `/roles/update-role`

Updates an existing role's name and description.

### Request

```json
{
  "role": "2290379f-667e-4c09-9f1e-be0b643ff327",
  "name": "PLEASE",
  "description": "The UPDATED THE BEST basic role"
}
```

### Response

```json
{
  "message": "Role updated successfully"
}
```

## List Roles

**Method**: `GET`  
**URL**: `/roles/list-roles`

Retrieves a list of all roles in the system.

### Request

```bash
GET /roles/list-roles
```

### Response

```json
[
  {
    "id": "2290379f-667e-4c09-9f1e-be0b643ff327",
    "name": "basic",
    "description": "The most basic role",
    "permissions": ["auth.accounts.list"]
  },
  {
    "id": "3409379e-8b79-5b93-a122-a4e0b654ff512",
    "name": "admin",
    "description": "Administrator role",
    "permissions": ["auth.accounts.create", "auth.accounts.delete"]
  }
]
```

## Describe Role

**Method**: `POST`  
**URL**: `/roles/describe-role`

### Request

```json
{
  "role": "2290379f-667e-4c09-9f1e-be0b643ff327"
}
```

### Response

```json
{
  "id": "2290379f-667e-4c09-9f1e-be0b643ff327",
  "name": "basic",
  "description": "The most basic role",
  "permissions": ["auth.accounts.list"]
}
```

## Delete Role

**Method**: `POST`  
**URL**: `/roles/delete-role`

Deletes a role from the system.

### Request

```json
{
  "role": "basic"
}
```

### Response

```json
{
  "message": "Role deleted successfully"
}
```

## Assign Roles

**Method**: `POST`  
**URL**: `/roles/assign-roles`

Assigns one or more roles to a user.

### Request

```json
{
  "account": "best@email.com",
  "roles": ["viewer", "admin"]
}
```

### Response

```json
{
  "message": "Roles assigned successfully"
}
```

## Remove Roles

**Method**: `POST`  
**URL**: ` /roles/remove-roles`

Removes one or more roles from a user.

### Request

```json
{
  "account": "newd@email.com",
  "roles": ["viewer"]
}
```

### Response

```json
{
  "message": "Roles removed successfully"
}
```
